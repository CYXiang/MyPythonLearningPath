# Python正则表达式学习指南

正则表达式是处理文本的强大工具，能够进行复杂的模式匹配和文本操作。本指南将帮助您掌握Python中正则表达式的使用。

## 学习目标

- 理解正则表达式的基本语法
- 掌握Python的re模块
- 能够使用正则表达式解决实际问题

## 内容结构

1. 正则表达式基础
2. Python中的re模块
3. 常用正则表达式模式
4. 实际应用案例

## 开始学习

请查看本目录下的各个子章节，按顺序学习：

- `01_regex_basics.py`: 正则表达式基础语法
- `02_re_module.py`: Python re模块详解
- `03_regex_patterns.py`: 常用正则表达式模式
- `04_practical_examples.py`: 实际应用案例
